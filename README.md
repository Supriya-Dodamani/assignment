# My Web App

## Overview
This project is a web application that provides user authentication features, including login, registration, password recovery, and a user dashboard. It utilizes Bootstrap for responsive design and includes custom styles and JavaScript functionality.

## Project Structure
```
my-web-app
├── src
│   ├── index.html          # Main entry point of the application
│   ├── login.html          # Login page
│   ├── register.html       # Registration page
│   ├── forgot-password.html # Forgot password page
│   ├── reset-password.html  # Reset password page
│   ├── dashboard.html      # User dashboard
│   ├── css
│   │   └── styles.css      # Custom CSS styles
│   └── js
│       └── app.js          # JavaScript functionality
├── package.json            # NPM configuration file
└── README.md               # Project documentation
```

## Setup Instructions

1. **Clone the repository:**
   ```
   git clone <repository-url>
   cd my-web-app
   ```

2. **Install dependencies:**
   ```
   npm install
   ```

3. **Run the application:**
   Open `src/index.html` in your web browser to view the application.

## Features
- User authentication with login and registration forms.
- Password recovery and reset functionality.
- User dashboard for account management.
- Responsive design using Bootstrap.

## Customization
- Modify `src/css/styles.css` to customize the appearance of the application.
- Update `src/js/app.js` to add or modify JavaScript functionality.
